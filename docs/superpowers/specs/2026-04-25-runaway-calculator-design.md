# 跑路计算器 - 设计文档

## 概述

单页面"跑路计算器"应用，帮助用户根据存款目标、已有存款、每月存款额和下班时间，计算距离"财务自由跑路"还有多久，以及每天距离下班还有多久。采用温暖活泼风格（粉色渐变），带趣味文案。

## 技术选型

- **单文件 HTML**：所有 HTML、CSS、JavaScript 写在 `index.html` 中
- **无外部依赖**：纯原生实现，无框架、无第三方库
- **数据存储**：使用 `localStorage` 持久化表单数据

## 页面结构

### 页面1：表单页

- 居中卡片布局，带 `#f093fb → #f5576c` 渐变背景
- 5个输入字段：
  - **目标存款**（数字输入，单位：元）
  - **已有存款**（数字输入，单位：元）
  - **每月存款**（数字输入，单位：元）
  - **上班时间**（time 类型输入）
  - **下班时间**（time 类型输入）
- 按钮："冲！💨"
- 表单验证：5个字段均为必填，目标存款 > 0，每月存款 > 0，上班时间早于下班时间

### 页面2：结果页

- **顶部居中**：趣味文案区域（根据状态动态显示）
- **中间核心区域**：
  - "最终跑路时间" 标签
  - 大字体显示：最终跑路年月（如 `2028年06月`）
  - 剩余天数（如 `还剩 823 天`）
- **下方圆环进度条**：
  - SVG 圆环进度条，显示今日下班倒计时
  - 中心显示：`HH:MM:SS` 倒计时
  - 底部标签："下班倒计时"
- **右上角**：⚙️ 设置按钮，点击返回表单页修改参数

## 计算逻辑

```
需要月数 = (目标存款 - 已有存款) / 每月存款（向上取整到月）
跑路日期 = 当前日期 + 需要月数
剩余天数 = 跑路日期 - 今天（向下取整到整天）

下班倒计时 = 下班时间 - 当前时间（时分秒实时递减）
```

## 状态与趣味文案

| 条件 | 文案 |
|------|------|
| 已有存款 >= 目标存款 | "还上啥班啊，跑路 🎉" |
| 当前时间 > 下班时间 | "别卷了，该走了 🏃💨" |
| 剩余天数 <= 30 | "快了快了，胜利在望！🔥" |
| 剩余天数 <= 365 | "坚持住，好日子不远了！💪" |
| 默认 | "冲！每一笔存款都是自由的基石 🚀" |

## 响应式设计

- **PC端**（≥ 768px）：卡片最大宽度 480px，居中显示
- **移动端**（< 768px）：全宽布局，padding 适配，字体适当缩小
- 圆环进度条自适应容器宽度

## 圆环进度条

- SVG 实现，外圈灰色轨道，内圈粉色渐变填充
- 进度 = 已工作时间 / 总工作时间（从设定的上班时间到设定的下班时间）
- 进度满格时，中心文案切换为"已下班！🎉"
- stroke-dasharray / stroke-dashoffset 动画实现

## 数据存储

localStorage 键名：`runaway_calculator_data`
存储内容：`{ targetAmount, currentAmount, monthlyAmount, onworkTime, offworkTime }`
- 表单页提交时保存
- 页面加载时读取并自动填充
